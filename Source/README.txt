Controls for Project 3
----------------------
A/a:   toggle attenuation
S/s:   toggle shader mode
T/t:   toggle textures on/off
W/w:   toggle wireframe
+/=:   decrease player FOV (zoom in)
_/-:   increase player FOV (zoom out)
P/p:   pause player movement
D/d:   toggle difficulty
x/ESC: quit the program
